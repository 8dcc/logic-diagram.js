AND gate

SR latch

Pulse S high to set, pulse R high to reset, then return both to 0 to hold the state.

D-latch

Set E=1 to make the latch transparent. Return E=0 to hold the stored value.

D flip-flop (rising-edge triggered)

Change D while CLK=0, then toggle CLK to 1. Q updates on the rising edge.
